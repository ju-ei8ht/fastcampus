### git   

```bash
mkdir [new folderName]
rm -rf [delete folderName]
git init
git remote add origin [repo addr]
git add .
git commit -m"[commit comment]"
git log --oneline
git revert [revert version]
git reset --soft/hard/mixed [reset version]
git push/pull origin [branch]
git branch -m [branch]
```
